# roadmap

planned features and improvements for taperd.

---

## near-term

### v1.1 — polish

- [ ] torn/ripped edge mode (irregular tear instead of clean zigzag)
- [ ] more built-in patterns: hearts, geometric, floral, holiday themes
- [ ] color presets / palette picker with popular washi tape colors
- [ ] save/load tape configurations as presets
- [ ] keyboard shortcuts for common actions

### v1.2 — web improvements

- [ ] true vector SVG export (canvas paths → SVG path elements)
- [ ] batch export (generate multiple tapes with different colors/patterns in one go)
- [ ] shareable URL with encoded tape settings (no backend needed)
- [ ] dark/light theme toggle for the web UI
- [ ] PWA support (installable, works offline)

---

## mid-term

### v2.0 — advanced tape

- [ ] curved/wavy tape strips (not just straight horizontal)
- [ ] tape wrapping around corners and objects
- [ ] multiple tape strips in one generation
- [ ] tape roll rotation animation in preview
- [ ] 3D perspective control (viewing angle slider)

### v2.1 — pattern studio

- [ ] in-app pattern editor (draw simple repeat tiles)
- [ ] pattern library with community submissions
- [ ] AI-generated patterns from text prompts
- [ ] pattern color remapping (recolor existing patterns)
- [ ] multi-color pattern support

---

## long-term

### v3.0 — ecosystem

- [ ] figma community library of pre-made tape components
- [ ] tape collection sharing (export/import tape packs)
- [ ] integration with other design tools (sketch, adobe XD)
- [ ] API for programmatic tape generation
- [ ] tape physics simulation (realistic peeling, stretching, folding)

---

## completed

- [x] core tape generation with 6 built-in patterns
- [x] figma plugin with live preview
- [x] custom image upload patterns
- [x] figma frame as pattern source
- [x] paper and rough texture overlays
- [x] finite roll physics
- [x] mask-based zigzag serrations (figma)
- [x] composited zigzag serrations (web)
- [x] transparent background exports
- [x] multi-format export (PNG, JPEG, SVG)
- [x] multi-scale export (1x–4x)
- [x] standalone web version
- [x] drag-to-pull tape interaction
- [x] roll z-order fix (spool renders on top of tape)
