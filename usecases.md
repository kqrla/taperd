# use cases

## who is taperd for?

### designers

- **stationery mockups** — add realistic washi tape to product shots, flat-lay compositions, and packaging mockups
- **scrapbook layouts** — layer tape strips across photos, cards, and ephemera for a handcrafted aesthetic
- **UI decoration** — use tape as a playful visual element in dashboards, onboarding screens, or empty states
- **brand assets** — generate on-brand tape with custom colors and uploaded logo patterns

### illustrators

- **journal spreads** — create digital bullet journal and planner pages with tape accents
- **sticker sheets** — design washi tape sticker packs for digital or print
- **pattern design** — preview how repeat patterns look on a curved 3D surface (the tape roll)

### content creators

- **social media graphics** — add tape strips to pin photos, notes, or quotes onto backgrounds
- **thumbnail decoration** — use transparent tape exports to overlay on video thumbnails
- **blog headers** — create themed tape banners with custom patterns

### educators & students

- **presentation slides** — tape elements onto slides for a craft/collage aesthetic
- **worksheet design** — use tape to highlight sections or frame content areas
- **art projects** — generate tape references for traditional media projects

### developers

- **placeholder art** — quickly generate decorative assets during prototyping
- **CSS/SVG study** — examine how the web version renders complex shapes with canvas compositing

## common workflows

### figma plugin workflow

1. open taperd in your figma file
2. pick a color that matches your design palette
3. choose a pattern (or upload your own)
4. adjust width to span across the element you want to "tape down"
5. click add to canvas
6. position the tape over the target element
7. the output is fully editable vectors — adjust colors, swap patterns, resize freely

### web export workflow

1. open web/index.html
2. design your tape (color, pattern, size)
3. enable transparent background
4. set export scale to 2x or 4x for high-res
5. export as PNG
6. import into any design tool, presentation, or document
7. the transparent background means it layers cleanly over any content

### batch creation workflow

1. design a base tape style
2. export at desired scale
3. change the color and/or pattern
4. export again
5. repeat to build a collection of coordinated tape strips
