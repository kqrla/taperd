# taperd

A Figma plugin for generating washi tape strips.

## features

- **animated spool** — rotates CCW and shrinks as you pull tape
- **finite / infinite roll** — toggle whether the roll depletes to the cardboard core
- **6 built-in patterns** — botanicals, animals, dots, stars, stripes, checks
- **custom image pattern** — upload an image or pick a Figma frame
- **over-dispensing visual** — tape exits from the top rim correctly
- **rip & add** — torn edge mode
- **add to canvas** — places the tape group in your Figma file

## controls

| control | description |
|---|---|
| width / height | tape strip dimensions |
| tape color | background fill |
| opacity | strip transparency |
| show roll | toggle the spool |
| finite roll | roll depletes to the cardboard core |
| repeat | tile the pattern |
| ink color | colour of built-in pattern icons |
| style | which built-in pattern |
| texture | none / paper / rough grain overlay |
| offset x/y | shift the pattern |
| scale | zoom the pattern |
| rotation | rotate the pattern |
